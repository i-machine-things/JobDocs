# Quote Module

Creates quote folders in the customer files directory.

## Order
Display order: 10 (first tab)

## Features
- Create quote folders with customer association
- Add files (blueprints and other documents)
- ITAR support
- Search and copy from existing quotes
- Convert quote to job

## Dependencies
- Shared utilities (utils, widgets)
- Core framework (base_module, app_context)

## UI File
- `ui/quote_tab.ui` - Main quote creation interface
