JobDocs v0.9.11

Job tracking and documentation for machine shops — Linux Flatpak repository.

Add this repository to Discover

Download and open the .flatpakrepo file — KDE Discover will prompt you to add it automatically.

Or from the terminal:

flatpak remote-add --user --from jobdocs https://i-machine-things-org.github.io/JobDocs/jobdocs.flatpakrepo

Then install:

flatpak install jobdocs io.github.i_machine_things.JobDocs
