WHAT IS JOBDOCS?
----------------
JobDocs helps you organize blueprint files and customer job folders.
Think of it as a filing system for your manufacturing or engineering business.


WINDOWS INSTALLATION - EASY METHOD
-----------------------------------
1. Download: JobDocs-Setup-0.2.0-alpha.exe
2. Double-click the file
3. Click "Next" a few times
4. Done! Look for the JobDocs icon on your desktop

If Windows shows a security warning:
  - Click "More info"
  - Click "Run anyway"
  (This is normal for new software)


LINUX INSTALLATION (DEBIAN/UBUNTU)
-----------------------------------
1. Download: jobdocs_0.2.0-alpha_all.deb
2. Double-click the file (Software Center opens)
3. Click "Install"
4. Done! Find JobDocs in your applications menu

Or from command line:
  sudo apt install ./jobdocs_0.2.0-alpha_all.deb


FIRST TIME SETUP
----------------
When you first open JobDocs, it will ask you to set up folders:

  • Blueprints Directory - Where you store blueprint files
  • Customer Files Directory - Where job folders will be created
  • ITAR Directories (optional) - For controlled/restricted files

Just click "Browse" and choose your folders.


PORTABLE VERSION (NO INSTALLATION)
-----------------------------------
Windows: Download jobdocs.exe and run it directly
Linux: Download jobdocs and run it directly

No installation needed - great for USB drives!


GETTING STARTED
---------------
After installation:

1. Open JobDocs (desktop icon or Start menu)
2. Complete the folder setup
3. Start creating jobs!

Use the tabs at the top to:
  • Create Job - Make new job folders
  • Create Quote - Make quote folders
  • Search - Find existing jobs
  • Add to Job - Add files to existing jobs

SYSTEM REQUIREMENTS
-------------------
Windows: Windows 10 or 11 (64-bit)
Linux: Ubuntu 20.04+, Debian 10+, or compatible

No other software needed - everything is included!


UNINSTALLING
------------
Windows:
  Start → Settings → Apps → Find JobDocs → Uninstall

Linux:
  sudo apt remove jobdocs
