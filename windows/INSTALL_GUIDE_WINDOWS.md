# How to Install JobDocs on Windows

**Simple 5-step guide for installing JobDocs**

---

## System Requirements

Before installing JobDocs, make sure your computer meets these requirements:

### Operating System
- **Windows 10** (64-bit) or newer
- **Windows 11** (any version)

### Minimum System Requirements
- **RAM:** 2 GB minimum (4 GB recommended)
- **Disk Space:** 200 MB free space for installation
- **Display:** 1024x768 screen resolution or higher

### Administrator Rights
- **Installation:** You'll need administrator rights to run the installer
  - If you're the main user of your computer, you likely already have this
  - The installer will ask for permission when you run it (click "Yes")
- **Running JobDocs:** No administrator rights needed after installation

### What's Included
- **Everything you need is included!**
- No need to install Python, PyQt6, or other dependencies
- JobDocs is a complete, standalone application

---

## Step 1: Download the Installer

1. Go to the JobDocs download page
2. Click on **"JobDocs-Setup-0.2.0-alpha.exe"** to download
3. Wait for the download to finish (about 60-100 MB)

**File location:** Usually downloads to your "Downloads" folder

---

## Step 2: Run the Installer

1. Open your **Downloads** folder
2. **Double-click** on "JobDocs-Setup-0.2.0-alpha.exe"
3. If Windows shows a security warning:
   - Click **"More info"**
   - Then click **"Run anyway"**

---

## Step 3: Follow the Installation Wizard

Click **"Next"** on each screen:

1. **Welcome Screen** → Click "Next"
2. **License Agreement** → Already accepted, click "Next"
3. **Choose Install Location** → Leave as default, click "Next"
4. **Ready to Install** → Click "Install"
5. **Completing Setup** → Check "Launch JobDocs", click "Finish"

**That's it! JobDocs will open automatically.**

---

## Step 4: Find JobDocs After Installation

JobDocs is now installed like any other Windows program:

### Option 1: Desktop Icon
- Look for the **JobDocs** icon on your desktop
- **Double-click** to open

### Option 2: Start Menu
- Click the **Start** button (Windows logo)
- Type **"JobDocs"**
- Click on **JobDocs** to open

### Option 3: Pin to Taskbar (optional)
1. Right-click the JobDocs icon on desktop
2. Choose **"Pin to taskbar"**
3. Now it's always visible at the bottom of your screen

---

## Step 5: First Time Setup

When you first open JobDocs, it will ask you to set up some folders:

1. **Blueprints Directory** - Where you keep your blueprint files
2. **Customer Files Directory** - Where job folders will be created
3. **ITAR Directories** (optional) - For controlled/restricted files

Click **Browse** to choose each folder, then click **OK**.

**You're ready to use JobDocs!**

---

## Need Help?

### How to Uninstall
1. Click **Start** button
2. Type **"Add or remove programs"**
3. Find **JobDocs** in the list
4. Click it, then click **"Uninstall"**

### Common Issues

**"Windows protected your PC" warning**
- This is normal for new software
- Click "More info" then "Run anyway"

**Can't find JobDocs after installing**
- Check your Desktop for the icon
- Or press Start and type "JobDocs"

**Program won't open**
- Right-click the JobDocs icon
- Choose "Run as administrator"
